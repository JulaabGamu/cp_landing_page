# Team Images

Place team member images in this directory with the following names:

- `member1.jpg`
- `member2.jpg`
- `member3.jpg`
- `member4.jpg`
- `member5.jpg`
- `member6.jpg`

## Image Guidelines

- **Format**: JPG, PNG, or WebP
- **Dimensions**: Square aspect ratio (recommended: 800x800px or 1000x1000px)
- **File size**: Keep under 500KB for optimal performance
- **Style**: Professional headshots or team photos work best

The images will be displayed in a responsive grid layout with rounded corners and hover effects.
